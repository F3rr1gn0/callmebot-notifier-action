# Publish Checklist

1. Rename repo public.
2. Add license.
3. Add `branding` to `action.yml`.
4. Add tests for inputs and channel mapping.
5. Run `npm install`, `npm run build`, `npm test`, `npm run typecheck`.
6. Create git tag `v0.1.0`.
7. Create GitHub Release.
8. Enable GitHub Marketplace listing.
9. Verify `uses: owner/callmebot-notifier-action@v0.1.0`.
